# 📊 Online Course Engagement Analysis

End-to-end Data Science and Business Intelligence project analyzing online course engagement using Python and Power BI.

---

## 📌 Project Overview

This project analyzes engagement data from an online learning platform to understand student behavior, completion patterns, and course performance.

The workflow combines **Python data analysis** with **Power BI visualization** to transform raw data into meaningful insights.

---

## 🔎 Data Processing

Data preprocessing was performed using **Pandas** including:

- Handling missing values
- Removing duplicates
- Feature transformation
- Data cleaning and preparation

---

## 📊 Exploratory Data Analysis

Key insights extracted include:

- Engagement trends across courses
- Course completion behavior
- User activity distribution
- Performance comparison between course categories

---

## 📈 Power BI Dashboard

The processed data was visualized in **Power BI** to build an interactive dashboard highlighting:

- Engagement trends
- Course completion rates
- User activity patterns
- Course category performance

---

## 🛠 Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

---

## 🎯 Business Impact

The analysis transforms raw engagement data into **actionable insights** that can support:

- Course optimization
- Marketing strategies
- Student retention improvement
